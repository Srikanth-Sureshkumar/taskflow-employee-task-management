# 🚀 TaskFlow – Employee Task Management System

A modern **MERN Stack Task Management Application** that helps managers assign tasks to employees, track progress, and improve team productivity through an intuitive dashboard.

---

## 📌 Project Overview

**TaskFlow** is a full-stack web application designed to simplify task management within a team.
Managers can create and assign tasks, while employees can view and update their task progress.

This system improves **workflow transparency, productivity, and collaboration**.

---

## ✨ Features

* 🔐 User Authentication (Login / Register)
* 👤 Role-based access (**Manager & Employee**)
* 📋 Create, view, and manage tasks
* 📊 Task status tracking

  * Pending
  * In Process
  * Completed
* 📅 Due date tracking
* 🗑 Task deletion
* 🎨 Responsive UI (Mobile, Tablet, Desktop)
* ⚡ Real-time dashboard view
